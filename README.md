# E-Prescription System

A full-stack web application for managing electronic medical prescriptions.

## Description
The system connects doctors, patients, and pharmacies in one platform to
create, view, and process medical prescriptions digitally instead of using
paper-based workflows.

## Tech Stack
- Backend: Java, Spring Boot
- Frontend: React, TypeScript
- Database: PostgreSQL

## Project Structure
- backend/ : Spring Boot backend application
- frontend/ : React frontend application

## Notes
This project was built for learning and demonstration purposes and focuses on
clean architecture and real-world full-stack development.
